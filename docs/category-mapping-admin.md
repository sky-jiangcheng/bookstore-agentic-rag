# 分类映射管理功能

## 快速开始

### 1. 访问管理界面

打开浏览器访问：
```
http://localhost:3000/admin
```

点击 **[分类映射管理]** 按钮打开设置对话框。

---

## 功能说明

### 映射管理 Tab

#### 筛选工具

| 筛选项 | 说明 | 默认值 |
|--------|------|--------|
| 🔍 搜索框 | 按分类名称搜索 | - |
| 🏷️ 馆别筛选 | 按适用馆别过滤 | 全部馆别 |
| 🎯 仅低置信度 | 只显示置信度 < 0.3 的分类 | 关闭 |
| 📊 筛选条件 | 最少书籍数、最高置信度 | 1000 本、0.3 |

**筛选条件说明**:
- **最少书籍数**: 只显示书籍数量 ≥ 该值的分类
- **最高置信度**: 只显示置信度 ≤ 该值的分类（用于找出需要审核的分类）

#### 数据表格

| 列名 | 说明 |
|------|------|
| 分类 | category 名称 |
| 书籍数 | 该分类下的书籍总数 |
| 适用馆别 | 映射的馆别列表（数组） |
| 置信度 | 自动计算的置信度（0-1） |
| 状态 | 自动分配 / 人工审核 |
| 操作 | 编辑、重算、重置按钮 |

#### 操作说明

**编辑**:
1. 点击 [编辑] 按钮
2. 在编辑面板中点击馆别按钮进行切换（选中/取消选中）
3. 点击 [保存修改] 或 [取消]

**重算**:
- 点击 [重算] 按钮
- 确认对话框
- 系统基于当前书籍分布重新计算映射关系

**重置**:
- 仅对"人工"状态的映射显示
- 点击 [重置] 按钮
- 删除人工映射，恢复为自动分配状态

### 质量监控 Tab

#### 汇总统计

| 指标 | 说明 |
|------|------|
| 总映射数 | `category_library_mapping` 表中的记录数 |
| 覆盖书籍 | 有映射关系的书籍总数 |
| 低置信度 | 置信度 < 0.3 且书籍数 > 1000 的分类数 |
| 总问题数 | 所有质量问题数量的总和 |

#### 问题分类

| 问题类型 | 说明 | 处理建议 |
|----------|------|----------|
| 🟡 低置信度映射 | 分类在多个馆别均匀分布 | 人工审核调整 |
| 🟠 未映射分类 | 存在于 books 表但不在映射表中 | 重新计算映射 |
| 🔴 馆别不匹配 | 书籍的 library_types 与映射不一致 | 批量同步或检查异常 |
| 🔵 孤立映射 | 映射存在但无对应书籍 | 删除孤立记录 |

#### 快捷操作

- **[查看低置信度分类]** - 跳转到映射管理 Tab 并应用筛选
- **[刷新数据]** - 重新加载质量统计数据

## 工作流程示例

### 场景 1: 人工审核低置信度分类

1. 打开分类映射管理对话框
2. 切换到 [映射管理] Tab
3. 打开 [仅显示低置信度] 开关
4. 查看筛选出的分类列表
5. 对每个需要调整的分类：
   - 点击 [编辑]
   - 勾选/取消勾选馆别
   - 点击 [保存修改]

### 场景 2: 检查数据质量

1. 打开分类映射管理对话框
2. 切换到 [质量监控] Tab
3. 查看各项问题统计
4. 如果有低置信度问题：
   - 点击 [查看低置信度分类]
   - 进入映射管理进行审核

### 场景 3: 重新计算单个分类

1. 在映射管理 Tab 中找到目标分类
2. 点击 [重算] 按钮
3. 确认操作
4. 系统基于最新数据重新计算映射关系

## API 参考

### GET /api/admin/category-mapping

获取映射列表

**参数**:
- `min_book_count` (可选): 最少书籍数，默认 1000
- `max_confidence` (可选): 最高置信度，默认 0.3
- `library_type` (可选): 馆别筛选
- `limit` (可选): 结果数量限制，默认 100

**响应**:
```json
{
  "mappings": [
    {
      "category": "长篇小说",
      "library_types": ["公共馆", "成人目录"],
      "confidence": 0.204,
      "book_count": 18157,
      "auto_assigned": false
    }
  ],
  "total": 1
}
```

---

### PATCH /api/admin/category-mapping

更新或删除映射

**请求体 (更新)**:
```json
{
  "category": "长篇小说",
  "library_types": ["公共馆", "成人目录"]
}
```

**请求体 (删除)**:
```json
{
  "category": "长篇小说",
  "action": "delete"
}
```

**请求体 (重算)**:
```json
{
  "category": "长篇小说",
  "action": "recalculate"
}
```

---

### GET /api/admin/category-quality

获取质量统计数据

**响应**:

```json
{

    "low_confidence": 41,
    "unmapped_category": 0,
    "mismatched_library": 15,
    "orphan_mapping": 0,

  }

```

## 最佳实践

### 审核优先级

1. **高优先级** - 书籍数 > 5000 且置信度 < 0.3
   - 长篇小说 (18,157 本)
   - 英语 (11,355 本)
   - 散文集 (8,611 本)
   - 童话 (5,116 本)

2. **中优先级** - 书籍数 1000-5000 且置信度 < 0.3
   - 中国文学、汉语、中国历史等

3. **低优先级** - 书籍数 < 1000
   - 可以暂缓处理

### 馆别分配原则

| 分类特征 | 建议馆别 |
|----------|----------|
| 儿童文学、童话、绘本 | 小学、公共馆 |
| 中学教辅、青少年文学 | 初高中、公共馆 |
| 专业书籍、学术著作 | 成人目录、大学 |
| 大众读物、畅销书 | 公共馆、成人目录 |
| 语言学习 | 根据难度分级 |

## 故障排查

### 问题：编辑后书籍列表没有更新

**原因**: 只更新了映射表，未同步到 books 表

**解决方案**:
```sql
-- 批量同步映射到书籍（谨慎使用，可能影响性能）
UPDATE books b
SET library_types = cm.library_types
FROM category_library_mapping cm
WHERE b.category = cm.category
  AND cm.auto_assigned = FALSE;
```

### 问题：置信度显示为 1.0 但书籍分布均匀

**原因**: 可能是数据尚未刷新

**解决方案**: 点击 [重算] 按钮重新计算该分类的置信度

### 问题：无法保存修改

**可能原因**:
1. 数据库连接问题
2. 权限不足（需要管理员权限）
3. 馆别数组为空

**解决方案**:
- 检查浏览器控制台错误信息
- 确认已登录且有管理员权限
- 确保至少选择一个馆别
