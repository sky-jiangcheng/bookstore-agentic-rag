## 筛选原则

### 保留条件

- 有明确书名、作者、价格、库存
- 优先保留有“内容简介”的书
- 题材尽量覆盖多个类别，适合测试检索与推荐多样性
- 库存大于 0

### 排除条件

结合 `书目筛选词.docx`，排除了以下噪声：

- 考研、教材、练习册、课堂、试卷等考试教辅
- 小学、初中、高中、少儿、绘本、注音、拼音等少儿/校园场景
- 明显不适合作为成人大众阅读推荐的强教学书目
